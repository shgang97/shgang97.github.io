《Spark官方文档》——快速入门

本文最后更新于：2022年5月27日 晚上

摘要：Spark 快速入门，交互式的 Spark shell 的使用，使用 Java 语言编写 Spark 应用程序。

快速入门

在翻译 Spark 官方文档 Quick Start 部分的基础上进行整理，原文中涉及 Java、Scala、Python 等语言，在翻译及操作演示过程中，优先选择 Java 语言，没有 Java 则选择 Scala，而其他的可以参看官方文档。

本教程是对 Spark 的一个简单介绍。首先，我们将通过交互式的 Spark shell（基于 Python 或 Scala）介绍 API，然后展示如何使用 Java、Scala 和 Python 编写 Spark 应用程序。

首先，请从 Spark website 下载 Spark 的发布版本，以便学习本指南。由于我们不会用到 HDFS，因此您可以下载适用于任何 Hadoop 版本的包。

请注意，在 Spark 2.0 之前，Spark 的主要编程接口是弹性分布式数据集（RDD）。在 Spark 2.0 之后，RDD 被 Dataset 取代，它像 RDD 一样是强类型的，但在底层进行了更丰富的优化。仍然支持 RDD 接口，您可以在 RDD programming guide 中获得更详细的参考。但是，我们强烈建议您改用 Dataset，它的性能比 RDD 更好。请参阅 SQL programming-guide 以获取有关 Dataset 的更多信息。
